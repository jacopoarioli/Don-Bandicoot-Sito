# Don-Bandicoot-Sito
Ecco una versione del README senza l'utilizzo delle linee orizzontali di separazione, mantenendo uno stile pulito e professionale.

🍎 Don Bandicoot
Benvenuti nel repository ufficiale di Don Bandicoot, un progetto nato dall'unione tra la letteratura classica spagnola e l'icona del gaming anni '90. 
Sviluppato con passione da un gruppo di studenti di informatica, questo sito web è il risultato di un progetto interdisciplinare volto a dimostrare come la letteratura italiana e straniera possa dialogare con lo sviluppo software.

🎭 Il Concept
Cosa succederebbe se Don Chisciotte della Mancia, invece di un ronzino, cavalcasse un destriero potenziato? E se i mulini a vento fossero in realtà temibili macchinari del Dr. Mulinex? Don Bandicoot rilegge i temi della follia, dell'idealismo e dell'avventura cavalleresca attraverso le lenti colorate del mondo di Crash Bandicoot. 
Il sito funge da hub per esplorare questa narrazione alternativa, offrendo approfondimenti critici e contenuti interattivi.

🛠️ Caratteristiche Tecniche
Il progetto è stato realizzato applicando le competenze acquisite nel corso di informatica:

Linguaggi: HTML, CSS, JavaScript.

Design: UI ispirata ai menu classici della saga di Crash, con un tocco cavalleresco.

Contenuti: Analisi comparativa tra il testo di Cervantes e il world-building dei videogiochi.

📚 Obiettivi Didattici
Questo progetto mira a colmare il divario tra materie umanistiche e tecniche:

Analisi Letteraria: Comprendere la figura dell'antieroe e la satira dei poemi cavallereschi.

Competenze Digitali: Gestione di un progetto web, dalla progettazione del layout alla pubblicazione.

Creative Problem Solving: Adattare concetti complessi come il "dualismo tra realtà e finzione" in un formato digitale moderno.
